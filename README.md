# Customer Churn Analysis Dashboard

## Project Overview

Customer retention is often more cost-effective than customer acquisition. This project analyzes customer churn data to identify at-risk customer segments, uncover key factors driving churn, and support data-driven retention strategies.

Using Power BI, DAX, and Excel, an interactive dashboard was developed to help stakeholders monitor churn trends, customer behavior, and business performance indicators.

## Objectives

* Analyze customer churn patterns and trends.
* Identify customer segments with high churn risk.
* Determine the main factors influencing customer attrition.
* Create an interactive dashboard for business users.
* Support retention and customer engagement strategies.

## Tools & Technologies

* Power BI

* Microsoft Excel

## Key Features

* Churn rate analysis
* Customer segmentation
* Interactive filters and slicers
* KPI monitoring
* Trend analysis
* Visual insights for decision-making

## Dashboard Insights

The dashboard helps answer questions such as:

* Which customer segments have the highest churn rates?
* What characteristics are common among churned customers?
* How does churn vary across different customer groups?
* What retention opportunities can be identified?
